Exponenciais Complexas – Visualização 3D

Este projeto apresenta uma visualização 3D interativa das exponenciais complexas, incluindo:

e^(jθ) (exponencial positiva)

e^(-jθ) (exponencial negativa)

Seno, cosseno e tangente derivados das exponenciais

Controles de velocidade, frequência e amplitude

Tema claro/escuro

Interação com mouse

Captura de tela integrada

Painel com valores ajustáveis em tempo real

A visualização foi desenvolvida usando Three.js e HTML/CSS puro.

Como usar

Basta abrir o arquivo index.html em qualquer navegador moderno.
Não é necessário servidor; tudo funciona localmente.

Conceitos abordados

Fórmulas de Euler

Relação entre exponenciais complexas e funções trigonométricas

Curvas paramétricas em 3D

Representação geométrica no plano complexo

Tecnologias utilizadas

Three.js para renderização 3D

CSS para estilo

JavaScript para animações e interatividad
